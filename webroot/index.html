Wi-Fi Tweaks for RN10 Pro

Antenna Configuration (2x2 MIMO)

This change allows the device's firmware to automatically select the best antenna configuration, enabling 2x2 MIMO for potentially double the throughput.

- gSetTxChainmask1x1=1
- gSetRxChainmask1x1=1
+ gSetTxChainmask1x1=0
+ gSetRxChainmask1x1=0

Transmit Power

Increases the transmit power of the Wi-Fi radio for a stronger signal and better range. This may increase battery consumption and heat.

- TxPower2g=10
- TxPower5g=10
+ TxPower2g=15
+ TxPower5g=15

2.4GHz Channel Bonding

Enables 40MHz channel bonding on the 2.4GHz band, which can significantly increase Wi-Fi speed.

- gChannelBondingMode24GHz=0
+ gChannelBondingMode24GHz=1
